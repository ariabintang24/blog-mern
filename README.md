# 📝 Go-Blog

Go-Blog is a modern **MERN stack (MongoDB, Express, React, Node.js)** blog platform that allows users to create, manage, and explore articles through a clean and intuitive interface.

The application provides a complete blogging experience, including content creation, user authentication, and profile management.

---

## ✨ Highlights

- ✍️ Create and manage blog posts (CRUD)
- 🔍 Search and filter blogs by category
- 👤 User authentication & profile management
- 🔐 Secure account features (update profile & password)
- 🎯 Clean, modern, and responsive UI

## 🚀 Features

### 🏠 Blog Listing & Categories
Users can browse blog posts displayed in a clean card layout, with category filtering such as Technology, Startup, Lifestyle, and Finance.

---

### 🔍 Search & Filter
Allows users to search blog posts by title and filter content by category for faster discovery.

---

### 📖 Blog Detail
Displays full article content in a structured and readable format.

---

### ✍️ Create & Manage Blogs (CRUD)
Authenticated users can:
- Create new articles
- Edit existing posts
- Delete their own content

---

### 👤 User Profile
Users can view and manage their profile information, including avatar and personal details.

---

### 🔐 Authentication & Security
- Login & Register system  
- Update profile information  
- Change password securely  

---


### 🎨 Responsive UI
Designed with a modern and clean layout, optimized for both desktop and mobile devices.

---

## 🛠 Tech Stack

**MERN Stack:**
- MongoDB — Database  
- Express.js — Backend framework  
- React.js — Frontend library  
- Node.js — Server environment  

**Other Tools:**
- Tailwind CSS — Styling  
- Vercel — Deployment
